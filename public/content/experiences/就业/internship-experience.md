**算法题**:

```javascript
// 实现一个LRU缓存
class LRUCache {
  constructor(capacity) {
    this.capacity = capacity;
    this.cache = new Map();
  }
  
  get(key) {
    if (!this.cache.has(key)) return -1;
    const value = this.cache.get(key);
    this.cache.delete(key);
    this.cache.set(key, value);
    return value;
  }
  
  put(key, value) {
    if (this.cache.has(key)) {
      this.cache.delete(key);
    } else if (this.cache.size >= this.capacity) {
      const firstKey = this.cache.keys().next().value;
      this.cache.delete(firstKey);
    }
    this.cache.set(key, value);
  }
}
```

## 实习工作内容

## 实习建议

### 面试准备
1. **算法基础**: 刷LeetCode，重点练习数组、字符串、树相关题目
2. **项目深度**: 对自己的项目要有深度理解，能回答各种细节问题
3. **技术广度**: 了解前端生态系统的基本概念
4. **表达能力**: 练习清晰、逻辑清晰地表达技术观点

### 实习期间
1. **主动学习**: 不懂就问，不要害怕暴露无知
2. **积极沟通**: 及时与导师和同事沟通进展和困难
3. **承担责任**: 对自己的代码负责，保证质量
4. **总结反思**: 定期总结工作经验和收获

### 职业规划
1. **明确目标**: 想清楚自己未来的职业方向
2. **持续学习**: 技术更新很快，要保持学习的习惯
3. **建立人脉**: 与同事和同行建立良好关系
4. **积累经验**: 在每个项目中都要有收获和成长

## 总结

这次实习经历让我对前端开发有了更深入的理解，也让我明确了自己的职业方向。在字节跳动的三个月，我不仅提升了技术能力，更重要的是学会了如何在职场中工作和生活。

对于想要进入大厂实习的同学，我的建议是：
1. 提前准备，打好基础
2. 积极参与项目，积累经验
3. 主动学习，不断提升
4. 保持积极心态，享受过程
